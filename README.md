# GPTStudy

Generate study material with the GPT AI

## Tech Stack

**Client:** Next.js, React, Tailwind CSS, Daisy UI

 
## Run Locally  
Clone the project  

~~~bash  
  git clone https://github.com/SumitNalavade/GPTStudy
~~~

Go to the project directory  

~~~bash  
  cd gpt-study
~~~

Install dependencies  

~~~bash  
npm install
~~~

Start the server  

~~~bash  
npm run dev
~~~
